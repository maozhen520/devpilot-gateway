### 前提要求

本项目编译或者运行测试需要 Java 17 或者更高版本。

### 项目构建

1. 将项目克隆到本地。
    `git clone https://github.com/openpilot-hub/devpilot-gateway.git`
2. 运行如下指令构建项目。
    `mvn clean package`
3. 运行如下指令启动项目。
    `java -jar target/devpilot-gateway-1.0.0.jar`
